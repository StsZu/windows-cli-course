# Windows CLI Course

Практичний курс команд CLI для **Microsoft Windows** українською: PowerShell, CMD, WSL, SSH, Git, winget.

**Головна мета — не вивчити всі команди Windows, а навчитися швидко знаходити потрібну команду, розуміти її ризик і застосовувати її в реальному сценарії.**

## Швидкий старт

```powershell
start index.html
```

GitHub Pages: **https://stszu.github.io/windows-cli-course/**

## Модулі тренажера

1. Windows Terminal basics
2. PowerShell basics (cmdlets, alias-и)
3. CMD basics
4. Файли та папки
5. Пошук і текст (Select-String)
6. Мережева діагностика
7. SSH / MikroTik / Raspberry Pi
8. Git і GitHub
9. Python, Node, winget, choco
10. Процеси, сервіси, порти
11. WSL Linux
12. Небезпечні команди
13. 14-денний план практики

## Оболонки

- **PowerShell** — основна сучасна CLI
- **CMD** — класична оболонка
- **WSL** — Linux всередині Windows
